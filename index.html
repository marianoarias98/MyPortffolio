Preguntas Frecuentes

¿Qué incluye el plan básico?

+

  • Sitio web funcional.
  • Diseño receptivo.
  • Carga de contenido.
  • Blog
  • Hasta 3 páginas.
  • 3 revisiones.
  • Dominio personalizado
  • Hosting.
  • Certificación SSL.
  • Tiempo de entrega 2-4 días.

No se incluye:

  • Edición del contenido: de ser necesario se deberá abonar como mantenimiento del sitio web.

¿Cuál es el precio?

+

El precio actual del desarrollo de tu web profesional con el plan básico es de AR $10000. En caso de necesitar algo mas especifico o desarrollado el precio varía.

¿Qué se necesita para empezar?

+

Una vez contactes conmigo se requerirá todo el contenido de texto, imágenes, video y lo necesario por adelantado como así también las indicaciones básicas para el diseño de la web.

¿Incluye revisiones?

+

Se incluye en el plan básico hasta 3 revisiones

¿Cúal es el metodo de pago?

+

Se puede abonar el pago con tarjeta de credito, tarjeta de debito y pagos en efectivo con la plataforma de Mercado Pagos.
